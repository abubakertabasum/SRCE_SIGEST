Formatting Toolbar

FormatFormats the displayed source code and displays the reformatted code.  The view territory may be expanded or decreased by using the cascade and tile buttons in the 'Window' menu.

Note:  Formatter Plus always compares each output file to the input file.  If it were to ever find differences other than spacing, new lines, or lowercase-UPPERCASE differences, it would not format the file.  Be assured that Formatter Plus will never alter your valuable files unless the reformatted output files are completely compatible with their respective input files.

CheckAllows the user to check the source code for syntax errors without formatting the code.  If there are no syntax errors found, a green button and the words 'No Errors' will appear in the lower right hand corner of the active window.  If an error is found the line with the error will be highlighted with a yellow arrow to the left of the line.  A red light and the words 'Syntax Error' will occur in the lower right hand corner of the active window.  These errors will need to be fixed before running the code.
ReviewInstantly analyze your code for correctness, maintainability, efficiency, readability, and program structure. 
ProfileDisplays a quantitative analysis of the source code, including a breakdown on PL/SQL version dependencies and characteristics of your SQL usage and modularization.  
Find In FilesSearches text in specified files for the next reference of the supplied text string.  Results are displayed in the Output bar.
Search KBaseDisplays a drop down menu allowing the user to search the Knowledge Base by keyword, or launch the knowledge base directly into the PL/SQL Development topics or Advisor Topics.
