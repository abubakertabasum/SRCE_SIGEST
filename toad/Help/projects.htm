Formatter Plus allows the user to group files in one convenient easy to use file called a "Project".  This section of the user's guide will instruct the user on creating a project.

Creating a Project

From the menu bar click on Project and then New.  A pop-up dialog box will appear asking you for the name of the new project and the location in which to save your new project.  Once you have entered the required information in this dialog box, your Project Bar will change to look like the one below.

This is your new project.  The sections below will outline how to create folders within the project, insert files into the project, and save a project.

Creating Folders within a Project

To create new files within a Project right click in the Project Window.  Select Create New Folder.  A properties dialog box will appear in the middle of the screen.  

In the first field of the dialog box enter the Folder Name.  In the second field, specify the types of files to be included in this folder.  You can check the Include Subfolder box depending on whether or not you want to include files from subfolders in the newly created folder.  Press Enter and the new folder will appear in the Project Bar.  You can also select Create New Folder from the Project option of the Menu Bar.

Inserting Files

There are several ways to insert files into a project.  In all cases, the folder you want to insert files into must be selected. Note: Non-text files type files will not be processed and cannot be displayed. If you still wish to add them to the project, uncheck the source code option under properties for that file.

  1. From the Project menu on the menu bar select Insert Files.  This will bring up a dialog box allowing you to select the files you wish to insert into the new Project.

  2. Right mouse click in the project window and select Insert File(s) .  This will bring up the same dialog box allowing you to select your files.

Once you have inserted your files, the Project bar will change to reflect the files you inserted.  It will look something like the figure below.

Saving a Project

Finally, you will want to save the project you have created.  There are three ways to save a project:

  1. Click on the Save Icon in the standard toolbar.  This action will save the project again under the same name and in the same place as previous saves.  
  2. From the menu bar, select Project and then Save.  The same actions will occur as in  method 1 above.
  3. From the menu bar, select Project and then Save Project As.  This action will activate a pop-up dialog box asking for the project name and the location in which you would like to save the project.  

 Note:  Choosing Save from File on the menu bar will only save individual files - NOT entire projects.

You are now ready to format, review, and profile your code.
