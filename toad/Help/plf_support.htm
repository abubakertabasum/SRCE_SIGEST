The technical support staff at RevealNet is ready to assist you with any questions or difficulties you may encounter with our products. Below is information on how to contact our technical support team.

 

World Wide Web: http://www.revealnet.com/Support/support.htm
E-mail:   support@quest.com

To check whether you have the most recent version of Formatter Plus, please check the RevealNet Labs home page at http://www.revealnet.com, or call technical support at 949-754-8000.

When reporting problems, please include the following information:

  1. Complete description of the problem.
  2. Is the problem reproducible? If so, how?
  3. What version of Windows are you running? For example, Windows 95, Windows NT, etc.
  4. What version of Formatter Plus are you running? Select About Formatter Plus from the Formatter Plus Help pull-down menu.
  5. What version of PL/SQL are you running?
  6. If a dialog box with an error message was displayed, please include the full text of the dialog box, including the text in the title bar.
  7. If possible, please include a copy of the failing code for use with recreating the problem.
