Right Mouse Options for the Project Window

Displays a dialog box allowing the user to choose the files to insert into the current project. Files can be inserted 3 different ways.
  1. Under File Extension Filter, enter the extension of the files you wish to include in the project (ex. sql). Click the Insert Button. This will insert all files with a particular extension into the project.
  2. Drag files and or folders from the pop up dialog box into the project window.
  3. Select the files you wish to insert and click the Insert button.
Creates a new subfolder in the current project.
Deletes the selected files from the current project.
This feature is only active you have used the drag and drop method for inserting files into your project.  If you have dragged a folder or file into your project and then edited the files outside of Formatter Plus, clicking on refresh will update the files in the Formatter Plus project to reflect the changes made outside Formatter Plus.
Formats the selected files or all of the files in the project.  If the project name is selected, all of the files in the project will be formatted.  If individual files are selected, they will be formatted.
Checks the syntax of the selected files or the entire project.  If the project name is selected, the entire project will be syntax checked..  If individual files are selected, they will be syntax checked. 
Selects everything in the current source code file.
Displays a statistical analysis of the main components of PL/SQL applications in the selected files.  If the project name is selected, an analysis of each file in the project will be displayed in the Output window.  
Displays a pop-up dialog box allowing the user to search for specified text in the project files.
Displays a date/timestamp list of your archives and the number of files in each. Clicking on an archive date displays the file name (with its full path ) that was archived. Currently only entire archives may be restored. Archived files are restored to their original location.
This option will create a backup of the selected file(s) and log it into the Formatter Plus archives list.
Displays as pop-up window showing the complete path of the current project or selected files.
