Right Mouse Options for the Edit Window

Formats the selected source code file.
Checks the selected source code file for syntax errors.  Error Free code results in a green light at the bottom of the status bar.  Errors produce a report in the Output bar.  The report details the position of the error in the code and recommendations for correcting the error.  The line of code containing the error is also highlighted in Yellow and marked with a yellow arrow to the left of the line.
Displays tips and recommendations for improving the selected source code file.
Displays a statistical analysis of the main components of the selected source code file in the output window.  The results are displayed in a grid in the output window.  Clicking on a component in the left portion of the output window displays that item's counts in the grid.  Clicking on a value in the grid displays detailed information about that item in the window below the grid.
Displays a pop-up window that allows the user to choose their formatting preferences.
This option will add the file in the current edit window to the open project. The file will be inserted into the project in the currently open folder. If no folder in the project is selected, the file will be inserted at the main project level. 
Note: Non-text type files will not be processed and cannot be displayed. If you still wish to add them to the project, uncheck the source code option under properties for that file.
Displays a pop-up dialog box allowing the user to search for related topics in RevealNet for Active PL/SQL Development.
Removes the selected text from the source code and pastes it into the windows clipboard.  You must have text selected before the cut function in activated. 
Copies the selected information from the source code window to the windows clipboard.  You must have text selected before the Copy function is activated.
Pastes information from the Windows clipboard into the open source code window of Formatter Plus.  There must be information on the Windows clipboard in order for the Paste function to be active.
Selects everything in the current source code file.
Closes the current open file in the edit window.
