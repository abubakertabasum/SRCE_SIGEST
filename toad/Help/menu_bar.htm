Text Editor

Edit Functions Displays a sub-menu with the following options:
 

Undo

Reverse changes made to code

Redo

Redo changes made to code
Revert Restore the source code to the state in which it was last saved.  This option is only valid for input from source code files directly opened by Formatter Plus.  It cannot be used to restore changes made to code either manually entered or cut and pasted into Formatter Plus.

Cut

Cut selected text to the Windows clipboard

Copy

Copy selected text to the Windows clipboard

Paste

Paste text in Windows clipboard into code window

Delete

Delete selected text

Select All

Selects all text in the current window 

Find

Searches text in the current window for the supplied text string

Find Next

Searches text in the current window for the next reference of the supplied text string

Find Previous

Searches text in the current window for previous references of the supplied text string

Replace

Replaces text in the current window with supplied new text

Format Text

Formats the source code file
Syntax Check

Checks source code for syntax errors
Code Review

Displays recommendations for improving the selected code files.
Code Profiler Toolbar displays a window with a quantitative analysis of the source code
Add File to Project

This option will add the file in the current edit window to the open project. The file will be inserted into the project in the currently open folder. If no folder in the project is selected, the file will be inserted at the main project level. 
Note: Non-text type files will not be processed and cannot be displayed. If you still wish to add them to the project, uncheck the source code option under properties for that file.

Options Menu

Formatting Options

Displays a dialog box allowing the user to set Code Formatting Options.  The recommendation options in the GUI do not affect the Code Advisor.

Set Options File

Displays a dialog box allowing the user to choose a file to save the formatting options in.  Multiple Formatting Option files can be set up.  To do this, set one set  of formatting options and save it.  The file will be called FmtPlus.opt.  Next, go through windows explorer, find and copy the file FmtPlus.opt and re-name it.  Then set formatting options for a second set of preferences and save them to the new file.  

Formatting options may be shared among multiple users if the options file is placed on a shared drive, such as a LAN server. Individual users must then choose that option file through the Set Options File dialog.

Note: If a user has RevealNet PL\Formatter already installed on their machine, it will be removed when they install Formatter Plus. However, the PL/Formatter options file will be saved as plfoptions.opt. This file may then be chosen through the Set Options File dialog for use with Formatter Plus.
Reset Action Scope The Action Scope defines whether to perform the requested action (Format, Syntax Check, Profile, or Analyze) against the file currently being edited or against the selected project entry.  This option resets the selection dialog so that you must choose the action scope.
