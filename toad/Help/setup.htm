  System Requirements

Formatter Plus is designed to work with:

PL/SQL releases 1.0 through 8.1.
Oracle SQL release 6.0 through 8.1
Oracle Forms release 4.5
SQL*Forms release 3.0
SQL*Plus release 6.0 through 8.1

Additionally, Windows 9x, Windows NT, Windows2000, or Windows ME is required for the GUI environment.
