Standard Toolbar

NewDisplays a new window to create source code in.  The view territory may be expanded or decreased by clicking and dragging the bottom right corner of the source code window or by using the cascade and tile buttons in the 'Window' menu.
OpenDisplays the Open dialog for selecting the source code input file to be displayed.  The input source code is displayed in the formatting window.  The view territory may be expanded or decreased by using cascade and tile buttons in the 'Window' menu.
SaveSaves the source code file under the existing file name.  If you DO NOT wish to overwrite the existing file with the formatted code, choose 'Save As' from the 'File' menu.
CloseCloses the current source window.  If the source code has been changed and not saved, a dialog box will pop up asking you if you wish to save the changes before closing.  You can respond yes, no, or cancel.  Choosing yes will save the changes to the source code before closing the window.  Choosing no will close the window without saving changes.  Choosing the cancel option will not close the window.
CutRemoves the selected text from the source code and pastes it to the windows clipboard.  You must have text selected before the cut icon is activated.
CopyCopies the selected information from the source code window to the Windows clipboard.  You must have text selected before the Copy icon is activated.
PastePastes information from the Windows clipboard into the open source code window in Formatter Plus.  There must be information on the Windows clipboard in order for the Paste button to be active.
UndoAllows the user to undo each successive action made since opening the source code.
RedoAllows the user to repeat each successive action made since opening the source code.
RevertRestores the original input source file.  This option is useful when changes have been made to the input source file within Formatter Plus and you do not wish to save these changes.

Note:  This option is only valid for input from source code files directly opened by Formatter Plus.  It cannot be used to restore changes made to code either manually entered or cut and pasted into Formatter Plus.
FindActivates a pop-up dialog box allowing the user to find specified text within the active source code window.  The user may set the specifications for the search.  The specifications include searching for whole word only, matching case and searching up or down in the text.
Repeat Find Repeats a search in the current window for the Next reference to the specified text entered in the Find dialog box.  This feature is only  activated after the Find feature has been used.
